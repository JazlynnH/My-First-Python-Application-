Lesson 1: Introduction to Python Variables

What is a Variable?

A variable is a container for storing data values in a programming language. In Python, you can create variables to hold different types of information, such as numbers, strings, or boolean values.

The 'msg' Variable

In this example 'msg' is a variable that stores a string value. It's commonly used to store messages or text.

Explanation

❁ By using the 'msg' variable to store the string: "Hello World". The 'print(msg)' statement can now be used to show the content of the 'msg' variable.
